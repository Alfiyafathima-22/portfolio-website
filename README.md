# 🌐 Personal Portfolio Website

A full-stack personal portfolio website built to showcase my skills, projects, and certifications.

---

## 🚀 Tech Stack

### Frontend
- React.js (Vite)
- HTML, CSS, JavaScript
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

---

## ✨ Features

- 🧑 About section
- 💼 Projects showcase
- 🏆 Certifications (dynamic from MongoDB)
- 📩 Contact form
- 📱 Fully responsive design
- 🌙 Dark mode support
